## Featured Projects

### AutoBlinds  
**ESP32 · Swift · BLE · KiCad · Onshape · PCB Design · 3D Printing**

An automated smart blind controller that attaches to existing chain-loop blinds and adjusts them using ambient light sensing, scheduling logic, and mobile app commands.

- Built ESP32 firmware and BLE communication for wireless device control.
- Developed a native Swift iOS app for controlling the embedded device.
- Designed custom PCB hardware in KiCad and CAD components in Onshape.
- Created a physical prototype using soldering, PCB assembly, STL export, and 3D printing.

### IoT Wardrobe Management Platform  
**FastAPI · Python · MySQL · ESP32 · MQTT · Docker**

A full-stack IoT platform for wardrobe monitoring, sensor telemetry, and AI-powered clothing recommendations.

- Built backend services, REST APIs, authentication, sessions, and database architecture.
- Integrated ESP32 devices and MQTT communication for sensor data collection.
- Developed frontend dashboard functionality for users, devices, and recommendations.
- Collaborated in a two-person team to deliver an end-to-end web, database, and IoT system.

### Python IDLE Editor Enhancements  
**Python · AST · Tkinter · Git**

A large-codebase software engineering project extending Python IDLE with editor features and performance improvements.

- Implemented code folding, fold gutters, placeholder rendering, and keyboard shortcuts.
- Improved responsiveness through AST caching, fold-state tracking, and redraw optimization.
- Developed unit and GUI tests for editor behavior and edge cases.
- Collaborated using pull requests, code reviews, issue tracking, and GitHub workflows.

### Ray Tracing Renderer  
**C++ · Computer Graphics**

A C++ ray tracing engine supporting realistic lighting, geometry intersection, and rendering effects.

- Implemented sphere and triangle intersections, anti-aliasing, soft shadows, recursive reflections, and global illumination.
- Built Lambertian diffuse shading and Blinn-Phong material models.
- Reduced rendering artifacts using radiance clamping and firefly reduction.
- Analyzed BVH acceleration as a path toward improved rendering performance.
